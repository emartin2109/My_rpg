# Epitech | myrpg
## Description of the project

The objective of this project is to develop a custom RPG from the ground up.

Please note that the project is currently incomplete, so you may encounter several bugs and crashes.
### Functionality:

* Ray casting
* Fighting mechanics
* CSFML
* Pathfinding
* Particles
* Custom map creation using text files
* Menus and various parameters

Main Bugs:

* Crash occurs when the "How to Play" button is pressed.
* Several memory leaks present.
* The fourth level cannot be completed with the current map.
